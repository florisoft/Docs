With this setting enabled, you can import SalesOrders into the stocks below that setting.
Optional: With this checkbox enabled, you can also have SalesOrders divided directly using the value specified in the nextLegIdentifier (seatnumber is included in the SalesOrder).
The value in the nextLegIdentifier searches for the seatnumber that is known from the plate numbers.
